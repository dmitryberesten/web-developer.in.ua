Пояснення HTML :)

Веб-сайт — це онлайн-ресурс, до якого можна отримати доступ через Інтернет. Веб-сайти — це документи на сторінці (одно- або багатосторінкові), які надають різну інформацію кінцевому користувачеві. Це означає, що веб-сайти створюються для різних цілей. У нас є веб-сайти електронної комерції, веб-сайти соціальних мереж, веб-сайти новин, веб-сайти електронного навчання, веб-сайти компаній та багато іншого.

Веб-сайт електронної комерції: зазвичай він спрямований на об’єднання продавців і покупців. Наприклад amazon.com, jumia.com, aliexpress.com.

Соціальний веб-сайт: це середовище для соціальної взаємодії між людьми з різних куточків світу. Наприклад Facebook.com, twitter.com, slack.com

Веб-сайт електронного навчання: це веб-сайт, який моделює систему класних кімнат для навчання людей на різних курсах за їхнім вибором. Наприклад udacity.com, coursera.com, udemy.com.

Основною характеристикою веб-сайту є те, що його URL-адреса містить префікс http або https і доменні імена, які закінчуються на .com, .co тощо.

Давайте разом подумаємо, навіщо html потрібен для веб-сторінок.

Спілкування з французом англійською буде невдалим і не сприйнятим, оскільки він не розуміє англійської. Браузери — це те, що відображає web-сторінки, і те, що браузери розуміють як мови розмітки, тому браузер повинен використовувати мову розмітки, щоб відобразити вміст документа.

Що ви повинні розуміти при написанні HTML-коду.

Семантичні елементи: це теги, які мають певне значення. Наприклад, тег <p> </p> означає створення абзацу, тег <h1> </h1> означає створення текстового заголовка. Однак теги <div> </div> або <span> </span> не надають конкретного значення вмісту. HTML SEMANTICS ELEMENT чітко передає своє значення як розробнику, так і веб-браузеру.

Давайте подивимося на теги, але детально.

Теги HTML можна класифікувати на контейнерні теги та порожні теги. Теги-контейнери — це прості елементи HTML, вони поділені на три основні частини: відкриваючий тег, вміст і закриваючий тег, тоді як порожні теги містять лише відкриваючий тег.
Приклади тегів HTML:
